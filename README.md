# Exoplanet-Collision-Risk-Predictor
Developed a machine learning model during NASA Space Apps Challenge to predict potential collision risks between exoplanets and Earth using NASA datasets. Achieved 94% accuracy through data preprocessing, feature engineering, and model optimization. Built using Python and scikit-learn. Project available on GitHub.
• Collaborated in a team during the NASA Space Apps Challenge to develop an AI-based solution using real NASA datasets  
• Designed and trained a machine learning classification model to assess potential collision risks between detected exoplanets and Earth  
• Performed data preprocessing, feature engineering, and model tuning to improve predictive performance  
• Achieved 94% prediction accuracy through model evaluation and validation techniques  
• Utilized Python with pandas, NumPy, and scikit-learn for data analysis and model development  
• Presented findings and technical approach as part of the competition submission  
• Project code and model implementation published on GitHub
